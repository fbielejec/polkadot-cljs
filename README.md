## keccak256 ##

### Development ###

```bash
cd scripts/keccak256
yarn deps
yarn watch
yarn start
```

Get the REPL:
```
M-x cider-connect-clojurescript
```

### Release ###

```bash
cd scripts/keccak256
yarn release
```
